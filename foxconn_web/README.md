#### 环境配置
- 安装`nodejs`

#### 安装
- `npm install`

#### 启动
- `npm run start`

#### 打包
- `npm run build`

#### 页面切换
- **路由控制**，通过在地址栏输入`smt`或`product`切换页面
- 1.环境看板：首页
- 2.板测看板：`smt`
- 3.成品测试看板：`product`
